---
title: "Update Settings"
description: "Updates the settings for the API customer."
api: "POST /settings"
---

## Body

<ParamField body="webhookUrl" type="string" required>
  The webhook URL to be stored on this customer's settings.
  It should be a valid URL.
</ParamField>

## Response

<ResponseField name="id" type="string" required>
  The settings ID.
</ResponseField>

<ResponseField name="webhookUrl" type="string" required>
  The customer's webhook URL.
</ResponseField>

<ResponseField name="webhookKey" type="string" required>
  The key used to authenticate webhook calls to the customer's app.
</ResponseField>

Example response:

```json
{
  "id": "<settings-id>",
  "webhookUrl": "https://server.domain.com/webhook/endpoint",
  "webhookKey": "<webhook-key>"
}
```
