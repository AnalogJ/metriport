---
title: "Delete User"
description: "Revokes access tokens for all providers and deletes the user."
api: "DELETE /user/delete"
---

## Query Params

<ParamField query="userId" type="string" required>
  The Metriport user ID to delete.
</ParamField>

## Response

<ResponseField name="message" type="string" required>
  Feedback on successful user deletion.
</ResponseField>

<ResponseExample>

```javascript Metriport SDK
import { MetriportDevicesApi } from "@metriport/api";

const metriportClient = new MetriportDevicesApi("YOUR_API_KEY");

await metriportClient.deleteUser("metriportUserId");
```

</ResponseExample>

```json
{
  "message": "User deleted successfully."
}
```
