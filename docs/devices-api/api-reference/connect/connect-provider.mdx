---
title: "Connect RPM Provider"
description: "Connects the user to a remote patient monitoring (RPM) device provider."
api: "POST /connect/rpm/:provider"
---

## Path Params

<ParamField path="provider" type="string" required>
  An RPM device provider. Currently, we only support Tenovi.
</ParamField>

## Query Params

<ParamField query="state" type="string" required>
  The user's [connect token](/devices-api/api-reference/user/create-connect-token).
</ParamField>

<ParamField query="deviceIds" type="string" required>
  A comma-separated string containing device ID(s).
<Info>Example of connecting two devices at once: `deviceIds=device-id-0001,device-id-1234`.</Info>
</ParamField>

<ParamField query="deviceUserId" type="string" required>
  The user ID (aka `Patient ID` for Tenovi) of the user that is connected to the device.
</ParamField>

<Info>
  Currently, [Tenovi](/devices-api/getting-started/tenovi) is the only RMP device provider we
  support.
</Info>

<ResponseExample>

```javascript Metriport SDK
import { MetriportDevicesApi } from "@metriport/api-sdk";

const metriport = new MetriportDevicesApi("YOUR_API_KEY");

const token = await metriport.getConnectToken("metriportUserId");

await metriport.connectProvider(
  "tenovi", 
  token, 
  "device-id-0001", 
  "85300"
);
```

</ResponseExample>
