## FHIR Resources:

The following resources are already supported for Data Contributions. More will be added in the future.

### Top-level Resources

- Bundle
- Composition
- Patient
- Encounter
- Practitioner
- Organization

### Data Resources

- DiagnosticReport
- Observation (categories: laboratory/vital-signs/survey/social-history)
- MedicationStatement
- Medication
- Condition
- AllergyIntolerance
- Location
- Immunization
- FamilyMemberHistory
