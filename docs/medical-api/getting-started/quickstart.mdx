## Overview

Getting started with our Medical API is pretty easy, you'll just need take the following steps:

1. [Create a developer account](#1-create-a-developer-account).
2. [Request API access](#2-request-api-access).
2. [Generate an API key](#2-generate-an-api-key).
3. [Integrate your app with Metriport](#3-integrate-your-app-with-metriport).

## 2. Request API access

On the Metriport developer dashboard go to the [Products & Billing page](https://dash.metriport.com/billing), click `Request Access` on the Medical API product,
and fill out the request form.

Once the form is submitted, we will review your 

review your request, reachout :
- Review the information
- Sign the BAA
- Grant you access to the Medical API
- Contact you via email with the signed doc stating that your request has been approved

After approval, when you visit the "Medical API" page on the dashboard, you should then be able to access the Medical API admin functionality.
