---
title: "Upload Document"
description: "Uploads a patient document to the server."
api: "POST /medical/v1/document/"
---

This endpoint uploads the specified file to our servers and returns a FHIR Document Reference.

## Query Params

<ParamField query="patientId" type="string" required>
  The ID of the Patient.
</ParamField>

## Body Fields

<ParamField body="file" type="File" required>
  The file that you would like to upload.
</ParamField>

<ParamField body="organizationName" type="string">
  The name of the organization that created this file.
</ParamField>

<ParamField body="practitionerName" type="string">
  The name of the practitioner who created this file.
</ParamField>

<ParamField body="description" type="string">
  The brief description of the file.
</ParamField>

## Response

A [DocumentReference](/medical-api/fhir/resources/documentreference) to the file you uploaded.
