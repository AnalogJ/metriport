---
title: "Add Patient Data"
description: "Add net new clinical information about a Patient."
api: "POST /medical/v1/patient/{id}/consolidated"
---

Add net new clinical data for a patient using a [FHIR Bundle](https://www.hl7.org/fhir/R4/bundle.html). This data will be shared back
with the HIEs to meet data contribution requirements.

<ParamField path="id" type="string" required>
  The ID of the Patient.
</ParamField>

## Body

<ParamField body="entry" type="array" required>
  The entry needs to be an array of [FHIR resources](/medical-api/fhir/resources).
</ParamField>

## Response

Returns a [Bundle](/medical-api/fhir/resources/bundle) of type "transaction-response" with the entry being the outcome of the request.
