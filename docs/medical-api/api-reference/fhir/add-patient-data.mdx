---
title: "Add Patient Data"
description: "Add net new clinical information about a Patient."
api: "POST /medical/v1/patient/{id}/consolidated"
---

Add net new clinical data for a patient using a [FHIR Bundle](https://www.hl7.org/fhir/R4/bundle.html). This data will be shared back
with the HIEs to meet data contribution requirements.

<ParamField path="id" type="string" required>
  The ID of the Patient.
</ParamField>

## Body

<ParamField body="entry" type="array" required>
  The entry needs to be an array of [FHIR resources](/medical-api/fhir/resources).
</ParamField>

<Info>
  At the moment there is a limit to how much data you can send in a single request. You can send up
  to 50 resources in a single request and the content-length must not exceed 1Mb.
</Info>

## Response

Returns a [Bundle](/medical-api/fhir/resources/bundle) of type "transaction-response" with the entry being the outcome of the request.

<ResponseExample>

```javascript Metriport SDK
import { Metriport } from "@metriport/api-sdk";

const metriportClient = new MetriportMedicalApi("YOUR_API_KEY");

const consolidatedData = await metriportClient.createPatientConsolidated("PATIENT_ID", {
  resourceType: "Bundle",
  type: "collection",
  entry: [
    {
      resource: {
        id: "1234",
        resourceType: "ServiceRequest",
        status: "active",
        intent: "directive",
        subject: {
          reference: "Patient/12345",
          display: "John Doe",
        },
        meta: {
          versionId: "12345",
          lastUpdated: "2023-05-24T13:21:28.527Z",
        },
      },
    },
  ],
});
```
