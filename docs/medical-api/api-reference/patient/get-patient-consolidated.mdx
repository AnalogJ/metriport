---
title: "Get Patient's Consolidated Data"
description: "Gets a Patient's Consolidated Data."
api: "GET /medical/v1/patient/{id}/consolidated"
---

## Path Params

<ParamField path="id" type="string" required>
  The ID of the Patient.
</ParamField>

## Query Params

<ParamField query="resources" type="string" optional>
  List of FHIR resources to be returned with the patient
</ParamField>

<Info>To view the list of available resources scroll to the bottom of the page.</Info>

## Response
